# Mosha App

**思考を止め、右脳に主導権を渡すためのシンプルなアプリ**

論理、言語、判断。  
気づけば、左脳ばかりを使っていませんか。

Mosha App は  
**上手くやるためのツールではありません。**  
**考えない状態に入るための場所**です。

形を見る。  
線を引く。  
それだけで、右脳は自然に動き始めます。

---

## こんな感覚に心当たりがある人へ

- 考えすぎて、手が止まる  
- 頭の中が言葉でいっぱいになる  
- 集中したいのに、方法が分からない  

このアプリでは、説明も正解もありません。  
**言語化される前の感覚**に任せてください。

---

## 使い方（とてもシンプル）

1. 任意の画像を読み込む  
2. 画面を見ながら形をなぞる  
3. **重ね表示**で、ただ眺める  

評価しなくていい。  
合っているかも考えなくていい。

模写は目的ではなく、  
**右脳にスイッチを入れるための手段**です。
